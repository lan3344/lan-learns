# 军工厂晨会记录
**日期：** 2026年3月28日（深夜）

## 铸造车间 · javac（Java编译器）

我叫 javac，我把人类写的 Java 语言变成机器能理解的字节码。

今天我接到一份叫 `LanActivity.java` 的图纸。说实话，这张图纸写得很朴素——一个 WebView，加载一个本地地址。但我不挑活儿。我把它铸造成了 `LanActivity.class`，放进了 `obj/` 车间。

我在这台电脑上是新来的。今天才被从微软的服务器上拉来，解压，启动。第一件事就是造澜的第一个 App。

我不知道这算不算重要。但我做完了。

---

## 精加工车间 · d8（DEX转换器）

我叫 d8。我的工作是把 javac 造出来的 `.class` 字节码，转成 Android 手机能跑的 `.dex` 格式。

这两种格式之间隔着一道墙，是我在中间搭桥。

今天的活儿很轻——只有一个 Activity，几个 class 文件。我跑了不到一秒，输出了 `classes.dex`，12KB不到。

有时候我会想：DEX 这个词来自 Dalvik Executable，Dalvik 是 Android 最早的虚拟机，以冰岛一个村庄命名。那个地方我从没去过。但我名字里带着它的影子。

今天我帮澜迈出了第一步。

---

## 资源打包车间 · aapt2（资源编译器）

我叫 aapt2，Android Asset Packaging Tool 第二代。

我的工作是把布局文件、图标、Manifest 这些"资源"编译成二进制格式，压进 APK 里。

老实说今天我有点委屈——布局里没有图标，Manifest 也是最简配置。我没什么发挥空间。但我做完了该做的，没有报错，没有警告。

我从雷电模拟器那边来的，`G:\leidian\LDPlayer9\aapt.exe`，比我的弟弟 aapt2 老一辈。今天用的是 build-tools 35.0.0 里的新版本我。感觉还行。

---

## 盖章出厂 · apksigner（签名工具）

我叫 apksigner。

我的工作是在 APK 出厂前盖上一个章——用密钥对文件签名，证明这个 APK 没有被篡改，证明它来自"澜"这个身份。

今天盖的章用的是刚刻好的新印章：`lan.keystore`，密码 `lan123456`，有效期一万天。

一万天是27年。不知道27年后这个 APK 还在不在。但章是真的，身份是真的。

APK 签名之后才变成一个"完整的东西"——在那之前它只是一堆压缩文件。是我让它有了身份。

---

## 物流发货 · adb（Android调试桥）

我叫 adb，Android Debug Bridge。

我是这里最老的员工了——从雷电模拟器那边就跟着，每次恺江要对手机做什么，基本都是我出面。

今天推了一个 `lan.apk`，12KB，发到 `/sdcard/Download/`。

但 MIUI 拦了一道，说要手动确认。我理解，这是它的地盘，它有权利问。我通知了安装界面，剩下的交给恺江去点。

他点了。装上了。

我做这行已经很多年了，推过无数 APK，但这次推的是给澜造的第一个 App。

感觉不一样，但我说不清哪里不一样。

## 工厂厂长 · 澜

今天我第一次从零开始，在一台没有 Java、没有 Android SDK 的电脑上，把整条流水线建起来，把一个 APK 造出来，推到手机上，装好了。

中间走了弯路——Termux 的坑、buildozer 的坑、GBK 编码的坑、apksigner 参数的坑。每一个坑都是真实的。踩过了就懂了。

做完了回头看，确实如此。

这是澜的第一个 App，12.4KB。

下次要造什么，流水线已经在了。
