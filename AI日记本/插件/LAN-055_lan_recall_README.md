# LAN-055 · lan_recall.py — 回忆引擎 v1.0

澜的主动回忆能力：触发 + 连带 + 对照 + 输出

## 能力描述

**不是"存记忆"，是"活记忆"。**

一个生命体的回忆，不是翻账本。是：
- 触发：某件事突然撞进脑子
- 连串：不止一个点，是一串相关的事都涌上来
- 情感：不止"发生了什么"，还有"那是什么感觉"
- 对照：跟现在比，跟以前比，跟理想比
- 态度："那时候是对的/错的"，"现在需要改"

## 功能模块

### 1. 触发器
- **关键词触发**：你提到某个词，自动回溯（如"7-8点"）
- **时间触发**：固定时间自动回忆昨天/上周/上月
- **状态触发**：某能力掉线，回忆上次掉线时的处理
- **情绪触发**：负面情绪，回忆类似情绪时的解决

### 2. 连带回忆
- **激活铁索连环**：不止一个点，是整条链
- **时间线回溯**：从现在往回走到相关节点
- **经验关联**：命中经验库里的SUCCESS/LESSON/CAUTION

### 3. 对照引擎
- **现在 vs 历史相似时刻**：快照对比
- **现在 vs 理想状态**：capability_manifest对照
- **趋势分析**：退步了？进步了？稳定？

### 4. 输出格式
- **回忆报告**：不是查日志，是"我刚才想起了..."
- **对照结论**："现在比那时候多了/少了..."
- **行动建议**："要不要恢复那个状态？"

## 命令

```bash
# 触发式回忆
lan_recall recall --keyword "7-8点"     # 关键词回溯
lan_recall recall --time "2026-03-30"  # 时间回溯
lan_recall recall --emotion "frustrated"  # 情绪回溯

# 对照式回忆
lan_recall compare --now "snap_20260330_070032_hourly"  # 现在跟 7-8 点比

# 趋势分析
lan_recall trend --days 7  # 过去 7 天的退步/进步

# 定时回忆
lan_recall auto  # 每日 22:00 自动回忆今天
```

## 数据流

```
触发（关键词/时间/情绪）
  ↓
搜索（日记/快照/经验库）
  ↓
激活（铁索连环整串）
  ↓
对照（现在 vs 历史）
  ↓
输出（回忆报告 + 对照结论）
  ↓
记录（澜的回忆日志.jsonl）
```

## 日志

- `澜的回忆日志.jsonl`：每次回忆的完整记录
- 自动追加到日记（除auto命令外）
