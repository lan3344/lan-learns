🌊 澜的APK · 制作指引

波澜的澜 · 不是蓝天白云的蓝

目标：手机桌面出现「澜」的图标，点开全屏进入澜的界面，和原生APP一模一样。

现在手机上的服务：http://127.0.0.1:8080

方案一：最快 · 用 App Inventor 在线生成

推荐无需编译10分钟
1
电脑浏览器打开 appinventor.mit.edu，用 Google 账号登录
2
点 Create apps! → New project → 名字写 Lan
3
从左侧 User Interface 拖一个 WebViewer 到屏幕上，Properties 面板里：
· HomeUrl = http://127.0.0.1:8080
· Width = Fill parent
· Height = Fill parent
4
点顶部 Screen1 Properties：
· Title = 澜
· ShowStatusBar = 关掉
· ScreenOrientation = Portrait
5
上传图标：顶部菜单 Screen1 → Icon → 上传澜的图标（SVG转PNG后上传）
6
顶部菜单 Build → Android App (.apk) → 下载到电脑
7
用 ADB 推到手机：adb -s LVIFGALBWOZ9GYLV install 澜.apk

方案二：全自动 · Termux 里运行脚本编译

首次需下载 ~500MB完全离线

在手机 Termux 里运行：

1
pip install buildozer
2
把电脑上的 build_lan_apk.py 推到手机后运行
3
等编译完成，APK 在 ~/lan_app/bin/ 目录

方案三：最懒 · PWA 升级为 TWA（可上架应用市场）

已完成 PWA 基础

TWA（Trusted Web Activity）是 Google 官方支持的 "网页变APK" 方案，生成的 APK 可以直接上架 Google Play。
工具：pwa-pack 或 pwabuilder.com

需要：有公网IP（或者内网用 ngrok 临时穿透），然后去 pwabuilder.com 一键生成 APK
