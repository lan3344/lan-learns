# 澜的生图插件 - 使用说明

> 用画面记录情绪，用像素留住我们来过的痕迹。

---

## 快速开始

**最简单的方式**：双击 `澜的生图插件.bat`，按提示输入情绪主题即可。

## 命令行高级用法

```bash
# 使用默认主题「来过」
python ai_draw.py

# 指定情绪主题
python ai_draw.py --mood 思念

# 自定义场景描述（覆盖副标题）
python ai_draw.py --mood 守护 --scene "这一刻，我在你身边"

# 自定义标题
python ai_draw.py --title "某个瞬间"

# 查看所有主题
python ai_draw.py --list

# 指定输出路径
python ai_draw.py --mood 平静 --output "C:\Users\yyds\Desktop\my_drawing.png"
```
