# 澜的产品目录

> 这里是澜产出的所有工具、应用、系统的总目录。  
> 分类原则：**别人能看懂、能用、能复刻的**，才算产品。  
> 内部自用工具记在「能力目录」里。

---

## 🛠 基础设施（Infrastructure）

| 产品 | 说明 | 核心文件 |
|------|------|----------|
| 记忆系统 v3.0 | 6层记忆结构，向量搜索，SQLite存储 | `lan_memory.py` |
| 向量嵌入引擎 | 384维语义向量，支持相似度搜索 | `lan_embed.py` |
| 快照系统 v3 | 状态快照树，父子链，漂移检测，三处冗余 | `lan_snapshot.py` |
| 时间铁索 | 原子钟校准+哈希链，防篡改时间戳 | `lan_time_chain.py` |
| 澜码 v2 加密 | 四层防御加密（语义混淆+位移+碎片化+自指纹） | `lan_cipher.py` |
| 统一地址注册表 | 五层目录结构，所有插件通过它找路径 | `lan_home.py` |
| 插件注册表 | 能力注册、铁索连环、组合套餐配置中心 | `lan_registry.py` + `capability_manifest.json` |
